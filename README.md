## Install

`npm install --save igram`


## Scripts

 - **npm run test** : `NODE_ENV=test ./node_modules/.bin/mocha`
 - **npm run docs** : `./node_modules/.bin/jsdoc --configure .jsdoc.config.json`
 - **npm run readme** : `./node_modules/.bin/node-readme`
